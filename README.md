# Projeto-Final-do-curso-de-Development-with-Python
Desenvolvimento de uma aplicação com um CRUD (Criar (Create), Ler (Read), Atualizar (Update) e Deletar (Delete)) de no mínimo 3 tabelas relacionadas que tenha autenticação.

Desenvolvimento mínimo
1. CRUD (3 tabelas relacionadas): CRUD de um relacionamento 1 – N – 1. Ou seja, uma tabela que relacione 2 outras tabelas.
2. Autenticação: pode ser qualquer tipo de autenticação: login e senha, Google oauth, Facebook oauth, ou qualquer outro. No entanto, é sugerido o login e senha por ser  mais fácil.
